# /restart — Claude Code Oturumunu Yeniden Başlat

Claude Code oturumunu yeniden başlatmak için aşağıdaki komutu çalıştır:

```bash
exit
```

Ardından terminalden tekrar `claude` komutu ile başlat.

---

**Otomatik yeniden başlatma (tek komut):**

```bash
exit && claude
```

Bu komut mevcut oturumu kapatır ve yeni bir oturum açar. MCP sunucuları ve ayarlar sıfırdan yüklenir.
