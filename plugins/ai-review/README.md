# AI Review Plugin

AI-powered code review plugin for Claude Code — automated review using configurable AI models.

## Features

- Automated code review on PR/commit
- Configurable review depth (quick/standard/deep)
- Output as PR comment or terminal report

## Requirements

| Requirement | Details |
|-------------|---------|
| Deps | `bash`, `gh` |

## Usage

Invoked via `/review` skill or triggered automatically on PR creation.

## Install

Installed as part of full claude-config setup via `./install.sh`.
