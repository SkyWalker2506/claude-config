# LinkedIn Outreach Guide

## Quick Reference

**Connection requests:** ~300 char limit; **no pitch** — relevance + curiosity. **InMail / messages:** Short; reference **public** activity (post, job change). **Voice notes** optional for high-touch — test with segment.

**Limits (approximate — verify current LinkedIn rules):** Weekly invite caps vary by account health; throttle new accounts hard.

| Step | Purpose |
|------|---------|
| Profile view | Signal (use ethically) |
| Engage on content | Warm before ask |
| Connect note | Why connect, no demo ask |
| DM after accept | Value + single CTA |

## Patterns & Decision Matrix

| Role | Angle |
|------|-------|
| Founder | Growth or capital efficiency |
| VP Ops | Process pain + benchmark |
| IC engineer | Technical respect, no buzzwords |

## Code Examples

**Connection note (under limit):**

```text
{{FirstName}} — saw your post on {{topic}}. We're exploring
{{narrow_problem}} with {{industry}} teams. Would be great to connect.
```

**Follow-up DM (post-accept):**

```text
Thanks for connecting, {{FirstName}}.

Quick context: we help {{persona}} reduce {{metric}} by doing {{one_liner}}.
If that's on your roadmap this quarter, happy to share how {{peer}} approached it.

If not, no worries — I'll stay useful in the feed.
```

**Tracking sheet columns (if not in CRM yet):**

```csv
profile_url,stage,last_touch_date,next_action,sequence_id
```

## Anti-Patterns

| Mistake | Risk |
|---------|------|
| Automation violating LinkedIn ToS | Account restriction |
| Copy-paste identical DMs | Reports + poor conversion |
| Pitching in connection note | Ignored / spam flag |
| Stalking private info | Creepy + compliance |
