# Validation Overrides — Golf Paper Craft

## Additional rules
- Tree/rock/spring/trampoline must not be placed on water/pit unless explicitly supported by an island/platform.
- Bridges must cross a meaningful gap/hazard.
- Hole/goal must be on stable ground and not accidentally overlapped by hazards.

## Bypass checks (must report)
- Direct flat max-power shot check
- High-arc max-tier shot check
- Multi-shot path existence when distance exceeds max single-shot envelope

## Required output fields (when proposing a level)
- Intended solve path
- Obvious wrong path + readable failure
- Purpose audit table
- Dependency model
- Validation section (spatial/gameplay/readability/difficulty + these overrides)
