## Methodology
1. Select 3-5 direct competitors (same category, similar audience)
2. Install and use each app for 15-30 min session
3. Score each dimension 1-5 using rubric below
4. Document with screenshots and annotations
5. Synthesize findings into actionable recommendations

## Scoring Rubric

| Score | Meaning |
|-------|---------|
| 5 | Best-in-class, sets industry standard |
| 4 | Above average, minor improvements possible |
| 3 | Meets expectations, nothing standout |
| 2 | Below average, noticeable gaps |
| 1 | Poor, significant usability issues |

## Evaluation Dimensions
1. **Onboarding** — time to first value, steps, friction
2. **Core Loop** — primary interaction flow efficiency
3. **Gamification** — engagement mechanics depth
4. **Visual Design** — consistency, modern feel, polish
5. **Accessibility** — contrast, touch targets, screen reader
6. **Performance** — launch time, transitions, responsiveness
7. **Retention Hooks** — notifications, streaks, rewards

## Vocab App Competitors
- Duolingo: gamification benchmark (streak, XP, leagues)
- Anki: SRS benchmark (spaced repetition depth)
- Quizlet: flashcard UX benchmark (flip, swipe)
- Memrise: multimedia learning benchmark
- Drops: visual/micro-session benchmark
