## Google Fonts URL (index.html için)

Tüm adayları yükleyen tek link:
```
https://fonts.googleapis.com/css2?family=Fredoka:wght@400;500;600;700&family=Luckiest+Guy&family=Patrick+Hand&family=Bungee&family=Lilita+One&family=Ranchers&family=Grandstander:wght@700,900&family=Nunito:wght@900&family=Comfortaa:wght@700&family=Quicksand:wght@700&family=Cinzel+Decorative:wght@700,900&family=Almendra+SC:wght@700&family=Orbitron:wght@700,900&family=Share+Tech+Mono&family=VT323&family=Russo+One&family=Baloo+2:wght@700,800&family=Pacifico&family=Bubblegum+Sans&display=swap
```

**Dikkat:** Bu URL sadece font seçim sürecinde index.html'de olmalı. Seçim yapıldıktan sonra sadece seçilen fontlar yüklenecek şekilde daraltılabilir (performans için).
