# Keyword Research Tools

## Quick Reference

| Araç sınıfı | Örnek kullanım |
|-------------|----------------|
| **Google Ads Keyword Planner** | Hacim tahmini (reklam hesabı gerekir) |
| **Search Console** | Gerçek sorgular, CTR, pozisyon |
| **Ahrefs / Semrush / Moz** | Rakip anahtar kelime, backlink (ücretli) |
| **AnswerThePublic / AlsoAsked** | Soru formatı, PAA genişlemesi |
| **Trends** | Mevsimsellik |

**Intent sınıflandırması:** Bilgilendirici / ticari araştırma / işlem — aynı kelime farklı intent taşıyabilir.

## Patterns & Decision Matrix

| Aşama | Çıktı |
|-------|--------|
| Seed list | Ürün, kategori, rakip marka |
| Genişletme | Long-tail, sorular, yerel varyant |
| Önceliklendirme | Hacim × rekabet × iş değeri |
| Haritalama | URL + içerik tipi |

### Kümeler

- **Pillar + cluster:** Ana sayfa / hub + destek makaleleri
- **Cannibalization:** İki URL aynı intent — birleştir veya netleştir

## Anti-Patterns

| Hata | Sonuç |
|------|--------|
| Sadece yüksek hacim | Dönüşüm yok |
| Otomatik çeviriyle keyword listesi | Doğal dil bozulur |
| Rakip markaya bid/SEO saldırısı | Marka hukuku riski |
