# Social Media Post Templates

## Quick Reference

| Format | İskelet | Platform |
|--------|---------|------------|
| **Problem → çözüm → CTA** | 3 cümle + link | LinkedIn, X |
| **Hook + liste** | Dikkat çekici ilk satır + 3 madde | X, Threads |
| **Hikaye** | Bağlam → ders → soru | LinkedIn |
| **Kanıt** | Metrik veya alıntı + yorum | Tümü |

**CTA:** Tek net eylem (yorum, kayıt, okuma) — çoklu CTA dağıtır.

## Patterns & Decision Matrix

| Amaç | Ton |
|------|-----|
| Bilinirlik | Hikaye / görüş — tartışma sorusu |
| Trafik | Özet + “tam metin” linki |
| Topluluk | Mention ve kullanıcı içeriği |

### Güvenlik ve marka

- Üçüncü taraf görsel — lisans kontrolü
- Mention ve hashtag — platform kuralları (spam eşiği)

## Anti-Patterns

| Hata | Sonuç |
|------|--------|
| Aynı metni her platforma | Algoritma ve format uyumsuzluğu |
| Link yığını | Düşük etkileşim |
| Jargon duvarı | Erişilebilirlik düşer |
