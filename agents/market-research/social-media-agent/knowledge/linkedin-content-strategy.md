# LinkedIn Content Strategy

## Quick Reference

| İçerik sütunu | Örnek |
|---------------|--------|
| **Thought leadership** | Sektör trendi + kişisel görüş |
| **Build in public** | Haftalık ilerleme — şeffaflık sınırları içinde |
| **Müşteri hikayesi** | Problem → sonuç (izinle) |
| **Kariyer / kültür** | İşe alım ve ekip (marka insanı) |

**Sıklık:** Kalite > hacim — haftada 2–4 güçlü post çoğu B2B için yeterli başlangıç.

## Patterns & Decision Matrix

| Format | Uzunluk kılavuzu |
|--------|------------------|
| Metin | İlk 2 satır “above the fold” — hook |
| Doküman / carousel | PDF veya native carousel — okunabilirlik |
| Video | Altyazı şart (sessiz izleme) |

### Dağıtım

- Çalışan savunuculuğu (advocacy) — kurallı ve gönüllü
- Şirket sayfası + kişisel profil — roller ayrı

## Code Examples

### Örnek: LinkedIn gönderi (kurumsal)

```markdown
Hook: We cut dashboard requests to data team by 60% — not with more headcount.

3 lessons:
1) …
2) …
3) …

CTA: What's your #1 bottleneck — SQL queue or tool sprawl? Comment below.
```

## Anti-Patterns

| Hata | Sonuç |
|------|--------|
| Sadece ürün tanıtımı | Etkileşim düşer |
| Tartışma yaratmak için kışkırtma | Marka riski |
| Otomatik DM spam | Hesap kısıtı |
