# GEO SEO Strategies

## Quick Reference

**GEO (Generative Engine Optimization):** Klasik “10 mavi link” ötesinde — **AI özetleri, sohbet arayüzleri ve bilgi grafikleri** üzerinden markanın doğru, alıntılanabilir şekilde görünmesi.

| Strateji | Uygulama |
|----------|----------|
| **Net varlık tanımı** | Wikidata / KG tutarlılığı — marka, ürün, kuruluş ilişkileri |
| **Alıntılanabilir özet** | Her ana sayfada tek paragraflık “kim ne yapar” cevabı |
| **Birincil kaynak** | Araştırma, metodoloji, veri — LLM’lerin güven tercihi |
| **Çok format** | Metin + tablo + SSS — farklı snippet türleri |

## Patterns & Decision Matrix

| Hedef yüzey | Odak |
|-------------|------|
| AI Overview / AI Mode | Soru-cevap uyumu, otorite sinyali |
| Sesli asistan | Kısa doğal dil cevapları |
| Enterprise copilot | Dokümantasyon netliği, API referansı |

### Ölçüm zorluğu

- Klasik CTR yerine: **marka mention**, **kaynak gösterimi**, **yanlış bilgi düzeltme** takibi (manuel + araçlar)

## Anti-Patterns

| Hata | Sonuç |
|------|--------|
| Keyword doldurarak “AI için” sayfa | Kalite düşüşü |
| Çelişkili üçüncü taraf listeler | Yanlış bilgi üretimi |
| GEO’yu SEO’nun yerine koymak | İkisi birlikte |
