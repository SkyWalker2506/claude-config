## 2. Bone Landmarks Visible Through Skin

These are critical for realistic sculpting as they anchor surface forms.

### 2.1 Head & Neck
- **Suprasternal notch (jugular notch)**: U-shaped depression at top of sternum (0, 0.03, 0.36)
- **Hyoid bone**: Slight protrusion at front of neck (0, 0.04, 0.40)
- **Mastoid process**: Behind the ear (+-0.04, -0.02, 0.44)
- **Zygomatic arch**: Cheekbone ridge (+-0.035, 0.02, 0.47)
- **Brow ridge (supraorbital margin)**: Above eye sockets (+-0.02, 0.04, 0.48)
- **Mental protuberance**: Chin point (0, 0.04, 0.44)

### 2.2 Shoulder & Chest
- **Clavicle**: S-curved bone, palpable along entire length (+-0.02 to +-0.10, 0.03, 0.36); first 2/3 cylindrical, last 1/3 flattens
- **Acromion process**: Bony point at top of shoulder (+-0.14, -0.02, 0.36)
- **Coracoid process**: Just below clavicle, anterior (+-0.10, 0.04, 0.35)
- **Sternum**: Central chest bone, manubrium to xiphoid (0, 0.04, 0.28-0.36)
- **Xiphoid process**: Tip of sternum (0, 0.04, 0.28)
- **Ribs (lower edge)**: Costal margin visible as inverted V (+-0.06, 0.03, 0.24-0.28)

### 2.3 Back
- **Spine of scapula**: Horizontal ridge on upper back (+-0.04 to +-0.10, -0.06, 0.32)
- **Medial border of scapula**: Vertical edge near spine (+-0.04, -0.06, 0.26-0.34)
- **Inferior angle of scapula**: Bottom tip, moves with arm (+-0.06, -0.06, 0.26)
- **Spinous processes (C7, T1)**: Most prominent vertebra bump at base of neck (0, -0.06, 0.38)
- **Sacral triangle**: Dimples at sacroiliac joint (+-0.04, -0.05, 0.02)

### 2.4 Arm & Hand
- **Medial epicondyle (elbow)**: Inner bony bump (+-0.15, 0.01, 0.22)
- **Lateral epicondyle (elbow)**: Outer bony bump (+-0.17, -0.01, 0.22)
- **Olecranon (elbow tip)**: Back of elbow (+-0.16, -0.03, 0.22)
- **Ulnar styloid**: Outer wrist bump (+-0.18, -0.01, 0.14)
- **Radial styloid**: Thumb-side wrist bump (+-0.17, 0.01, 0.14)
- **Knuckles (metacarpophalangeal)**: Visible ridges on fist back

### 2.5 Hip & Pelvis
- **Iliac crest**: Hip bone ridge (+-0.10, -0.02, 0.04); palpable from ASIS to posterior
- **ASIS (Anterior Superior Iliac Spine)**: Front hip point (+-0.09, 0.03, 0.03)
- **Greater trochanter**: Outer hip bump (+-0.12, 0, -0.02); aligns with wrist level
- **Pubic symphysis**: Central pelvis front (0, 0.04, -0.01)

### 2.6 Leg & Foot
- **Patella (kneecap)**: Front of knee (+-0.07, 0.03, -0.22)
- **Tibial tuberosity**: Below kneecap (+-0.07, 0.03, -0.24)
- **Tibial shaft (shin)**: Sharp anterior edge, no muscle cover
- **Medial malleolus**: Inner ankle bone (+-0.05, 0.01, -0.36); sits higher than lateral
- **Lateral malleolus**: Outer ankle bone (+-0.07, -0.01, -0.37); sits lower than medial
- **Calcaneus (heel)**: Posterior foot protrusion

## 3. Fat Distribution Patterns

### 3.1 Male (Android Pattern)
Fat accumulates centrally — abdomen, flanks, upper back:
- **Abdomen**: Primary depot; subcutaneous + visceral; obscures rectus abdominis
- **Love handles (flanks)**: Over iliac crest; softens waistline
- **Upper back**: Between scapulae
- **Neck/jaw**: Submental fat softens jawline
- **Chest**: Minimal subcutaneous fat over pectorals
- **Limbs**: Relatively lean; muscle definition visible longer

### 3.2 Female (Gynoid Pattern)
Fat accumulates peripherally — hips, thighs, buttocks, breasts:
- **Hips**: Over greater trochanter; widens hip silhouette significantly
- **Thighs**: Inner and outer; creates softer thigh-gap profile
- **Buttocks**: Gluteal fat pad; lifts and rounds glutes
- **Breasts**: Mammary fat pad over pectorals
- **Upper arms**: Posterior; softens triceps region
- **Lower abdomen**: Below navel; creates gentle curve
- **Cellulite visibility**: Higher due to estrogen; especially upper thigh/buttock

### 3.3 Sculpting Implications
| Body Area | Male Displacement | Female Displacement |
|-----------|------------------|-------------------|
| Shoulders | Sharp deltoid definition | Softer, rounder contour |
| Chest | Flat with pectoral ridge | Breast volume (varies) |
| Waist | Wider, straighter torso | Narrow waist, hourglass |
| Hips | Narrow, squared | Wide, rounded |
| Thighs | Defined quad separation | Smoother, fuller |
| Calves | Defined gastrocnemius | Softer contour |

## 5. Translation to Metaball/Displacement in Blender

### 5.1 Metaball Body Blockout (Phase 1)

Each major body volume maps to a metaball element:

```python
# Conceptual — E2 (Blender Script Agent) implements this
import bpy
from mathutils import Vector

def create_humanoid_blockout(scale=1.0):
    """Create metaball blockout for a standard humanoid."""
    mb = bpy.data.metaballs.new('Humanoid')
    obj = bpy.data.objects.new('HumanoidBlockout', mb)
    bpy.context.collection.objects.link(obj)
    
    mb.resolution = 0.15
    mb.threshold = 0.6
    
    # Element definitions: (name, x, y, z, radius, stiffness)
    elements = [
        # Head
        ("head",        0,    0,    0.46, 0.04, 2.0),
        ("jaw",         0,    0.02, 0.44, 0.025, 1.5),
        
        # Neck
        ("neck",        0,    0,    0.40, 0.02, 1.5),
        
        # Torso
        ("chest_upper", 0,    0.01, 0.32, 0.07, 2.0),
        ("chest_lower", 0,    0,    0.26, 0.06, 2.0),
        ("abdomen",     0,    0.01, 0.18, 0.055, 1.8),
        ("pelvis",      0,    0,    0.04, 0.065, 2.0),
        
        # Arms (right side, mirror for left)
        ("shoulder_R",  0.14, 0,    0.34, 0.035, 1.5),
        ("upper_arm_R", 0.16, 0,    0.28, 0.03,  1.5),
        ("forearm_R",   0.17, 0,    0.20, 0.025, 1.5),
        ("hand_R",      0.17, 0,    0.13, 0.02,  1.5),
        
        # Legs (right side, mirror for left)
        ("thigh_up_R",  0.07, 0,   -0.04, 0.045, 1.8),
        ("thigh_lo_R",  0.07, 0,   -0.14, 0.04,  1.8),
        ("calf_R",      0.07, 0,   -0.26, 0.03,  1.5),
        ("foot_R",      0.07, 0.03,-0.38, 0.02,  1.5),
    ]
    
    for name, x, y, z, radius, stiffness in elements:
        elem = mb.elements.new()
        elem.co = Vector((x * scale, y * scale, z * scale))
        elem.radius = radius * scale
        elem.stiffness = stiffness
        
        # Mirror left side
        if name.endswith("_R"):
            elem_l = mb.elements.new()
            elem_l.co = Vector((-x * scale, y * scale, z * scale))
            elem_l.radius = radius * scale
            elem_l.stiffness = stiffness
    
    return obj
```

### 5.2 Displacement for Muscle Definition (Phase 2)

```python
# Conceptual — E2 implements
def add_muscle_displacement(obj):
    """Add procedural displacement for muscle definition."""
    # Create Musgrave texture for muscle forms
    tex = bpy.data.textures.new('MuscleDefinition', type='MUSGRAVE')
    tex.musgrave_type = 'FBM'
    tex.noise_scale = 3.0        # Scale 2-5 for muscle-sized forms
    tex.dimension_max = 1.2
    tex.lacunarity = 2.0
    tex.octaves = 4.0
    
    # Add displacement modifier
    mod = obj.modifiers.new('MuscleDisplace', type='DISPLACE')
    mod.texture = tex
    mod.strength = 0.15          # Subtle; 0.1-0.3 range for muscles
    mod.mid_level = 0.5
    mod.direction = 'NORMAL'
    mod.texture_coords = 'LOCAL'
    
    return mod
```

### 5.3 Surface Detail via Multiresolution (Phase 3)

```python
# Conceptual — E2 implements
def add_skin_detail(obj, subdivisions=3):
    """Add multiresolution for tertiary skin detail."""
    # Add multiresolution modifier
    mod = obj.modifiers.new('SkinDetail', type='MULTIRES')
    
    # Subdivide to desired level
    for i in range(subdivisions):
        bpy.ops.object.multires_subdivide(
            modifier=mod.name,
            mode='CATMULL_CLARK'
        )
    
    mod.sculpt_levels = subdivisions
    mod.levels = subdivisions - 1   # Viewport can be 1 less
    mod.render_levels = subdivisions
    
    # Add fine noise for skin texture
    tex_pores = bpy.data.textures.new('SkinPores', type='VORONOI')
    tex_pores.noise_scale = 80.0   # Fine scale for pores
    tex_pores.distance_metric = 'DISTANCE'
    
    mod_pores = obj.modifiers.new('PoreDisplace', type='DISPLACE')
    mod_pores.texture = tex_pores
    mod_pores.strength = 0.02      # Very subtle
    mod_pores.mid_level = 0.5
    mod_pores.direction = 'NORMAL'
    
    return mod, mod_pores
```

## 6. Quick Reference: Sculpting Priority Checklist

1. **Silhouette first**: Get the overall proportions and big shapes right
2. **Bone landmarks**: Add clavicle, sternum, iliac crest, patella indications
3. **Large muscles**: Pectorals, deltoids, trapezius, latissimus, quadriceps, gluteus
4. **Secondary muscles**: Serratus, obliques, gastrocnemius, forearm group
5. **Fat/skin layer**: Soften transitions, add gender-appropriate fat distribution
6. **Tertiary detail**: Wrinkles, veins, pores (only at high subdivision)
