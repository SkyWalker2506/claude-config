# Proportion Systems

## 1. Head-Count Systems

The head is the universal unit of measurement in character design. The total height of a character expressed in "head heights" instantly communicates age, style, and power level.

### Standard Head-Count Chart

| Character Type | Heads Tall | Head-to-Body Ratio | Use Case |
|---------------|-----------|--------------------|---------| 
| Chibi / Super-Deformed | 2-3 | 1:1 to 1:2 | Cute mascots, emotes, icons |
| Toddler (age 1-3) | 4 | 1:3 | Child NPCs, young characters |
| Child (age 5-8) | 5-6 | 1:4 to 1:5 | Young heroes, cartoon protagonists |
| Stylized Teen | 6-6.5 | 1:5 to 1:5.5 | Anime/cartoon teens, light action |
| Realistic Adult | 7-7.5 | 1:6 to 1:6.5 | Realistic NPCs, civilians |
| Heroic Adult | 8 | 1:7 | Action heroes, warriors, adventurers |
| Superheroic | 8.5-9 | 1:7.5 to 1:8 | Gods, titans, imposing bosses |
| Elongated/Ethereal | 9-10 | 1:8 to 1:9 | Elves, celestial beings, horror |

### Fantasy Race Head-Count Map

| Race | Heads Tall | Notes |
|------|-----------|-------|
| Goblin | 3.5-4.5 | Big head (~25% of total height), short bowed legs, long arms |
| Halfling/Gnome | 4-5 | Adult proportions compressed into child-sized frame |
| Dwarf | 4.5-5.5 | Adult proportions, extra wide; torso = 2 heads, legs = 1.5-2 heads |
| Human (realistic) | 7-7.5 | Standard reference; torso = 3 heads, legs = 3.5-4 heads |
| Human (heroic) | 8 | Longer legs (+0.5 head), broader shoulders |
| Orc | 6-7 | Shorter than human but more massive; head is smaller relative to body |
| Elf | 8-9 | Elongated neck, longer limbs; head appears smaller |
| Troll | 5-7 | Highly variable; often hunched so standing height deceives |
| Dragon-kin | 7-10 | Depends on digitigrade legs; add tail as separate measurement |

## 2. Golden Ratio Applications

The golden ratio (phi = 1.618) appears naturally in appealing character proportions.

### Key Golden Ratio Breakpoints

| Measurement | Golden Ratio Application |
|------------|------------------------|
| Navel divides body | Total height / navel height = ~1.618 |
| Head to chin/navel | Chin-to-navel / head height = ~1.618 |
| Forearm to hand | Forearm length / hand length = ~1.618 |
| Finger segments | Each phalanx to next = ~1.618 ratio |
| Face thirds | Hairline-to-brow, brow-to-nose, nose-to-chin roughly equal |

### Practical Application

- When proportions "feel wrong" but you cannot identify why, check if the navel sits at roughly 61.8% of total height from the ground
- The shoulder-to-fingertip line should be roughly 1.618x the shoulder-to-elbow line
- For stylized characters, you can intentionally break the golden ratio to create visual tension — but know the rule before breaking it

## 3. Body Segment Proportions (8-Head Adult Reference)

### Vertical Division

| Landmark | Position (Head Units from Top) |
|----------|-------------------------------|
| Top of skull | 0 |
| Chin | 1 |
| Nipple line / mid-chest | 2 |
| Navel / elbow | 3 |
| Crotch / wrist | 4 |
| Mid-thigh / fingertips | 5 |
| Below knee | 6 |
| Mid-shin | 7 |
| Sole of foot | 8 |

### Width Proportions (Head-Width Units)

| Measurement | Width |
|------------|-------|
| Head width | 1.0 (reference unit) |
| Neck | 0.5-0.6 |
| Shoulders (male) | 2.0-2.5 |
| Shoulders (female) | 1.5-2.0 |
| Chest/ribcage | 1.5-1.8 |
| Waist (male) | 1.2-1.5 |
| Waist (female) | 1.0-1.2 |
| Hips (male) | 1.3-1.5 |
| Hips (female) | 1.5-2.0 |

### Limb Length Rules

| Limb Segment | Length |
|-------------|--------|
| Upper arm (shoulder to elbow) | 1.5 heads |
| Forearm (elbow to wrist) | 1.25 heads |
| Hand (wrist to fingertip) | 0.75-1.0 head (face height) |
| Upper leg (crotch to knee) | 2.0 heads |
| Lower leg (knee to ankle) | 2.0 heads |
| Foot length | 1.0-1.25 heads |

## 4. Muscle Group Placement Relative to Skeleton Landmarks

### Critical Bony Landmarks (Always Visible)

These landmarks remain visible regardless of muscle mass or fat — they anchor the anatomy:

| Landmark | Location | Why It Matters |
|----------|----------|---------------|
| Clavicles (inner heads) | Base of neck, front | Define neck-to-shoulder transition |
| Acromion process | Top of shoulder | Shoulder width point; deltoid origin |
| 7th cervical vertebra | Back of neck base | Prominent bump; neck-to-back transition |
| Spine of scapula | Upper back, diagonal | Trapezius dividing line |
| Iliac crest | Hip bones, side | Waist definition; oblique origin |
| Greater trochanter | Side of hip | Hip width point; leg rotation center |
| Medial epicondyle (humerus) | Inner elbow | Elbow landmark; forearm flexor origin |
| Olecranon | Back of elbow | Elbow point; always visible |
| Head of ulna | Outer wrist | Wrist landmark; wrist rotation reference |
| Patella | Front of knee | Knee center; always visible |
| Head of fibula | Outer knee, below | Lateral knee reference |
| Medial malleolus | Inner ankle | Ankle landmark (higher than outer) |

### Major Muscle Group Map

| Muscle Group | Origin | Insertion | 3D Form Impact |
|-------------|--------|-----------|----------------|
| **Trapezius** | Skull base to T12 spine | Clavicle + scapula spine | Diamond shape on upper back; defines neck silhouette |
| **Deltoid** | Clavicle + acromion + scapula spine | Mid-humerus (lateral) | Shoulder cap; 3 distinct heads visible in muscular characters |
| **Pectoralis major** | Sternum + clavicle + ribs | Upper humerus | Chest mass; runs UNDER deltoid (common mistake: deltoid under pec) |
| **Latissimus dorsi** | Lower spine + pelvis | Upper humerus (front) | V-taper; visible from front as armpit bulge |
| **Biceps** | Scapula (2 heads) | Radius (forearm) | Upper arm front mass; sits BELOW pectoralis (not over it) |
| **Rectus abdominis** | Pubic bone | Ribs 5-7 | "Six pack"; segmented by tendinous inscriptions |
| **External obliques** | Ribs 5-12 | Iliac crest + linea alba | Side torso; interdigitates with serratus anterior |
| **Gluteus maximus** | Ilium + sacrum | Femur (IT band) | Largest muscle; defines hip shape from side/back |
| **Quadriceps** | Pelvis + femur | Patella (via tendon) | Front thigh mass; 4 heads create surface detail |
| **Gastrocnemius** | Femur (back of knee) | Calcaneus (heel) | Calf shape; diamond form in back view |

### Common Anatomy Mistakes in 3D

| Mistake | Correct |
|---------|---------|
| Biceps overlapping pectoralis | Pec inserts OVER biceps at humerus |
| Deltoid inserted too high on arm | Deltoid tuberosity is at mid-humerus |
| Flat back without scapula movement | Scapula slides and rotates with arm movement |
| Symmetrical abs | Abs are often slightly offset; asymmetry is natural |
| Calf muscles at ankle | Calves are in the upper 2/3 of the lower leg |
| Missing serratus anterior | Finger-like muscles between ribs and scapula — visible in lean characters |

## 5. Style-Specific Proportion Adjustments

### Realistic to Stylized Spectrum

| Adjustment | Realistic | Semi-Stylized | Full Stylized |
|-----------|-----------|---------------|---------------|
| Head size | 1/7.5 of height | 1/6 of height | 1/4 to 1/5 of height |
| Hand size | = face height | +10-20% | +30-50% |
| Eye size | 1/5 of face width | 1/4 of face width | 1/3 of face width |
| Shoulder width | 2x head width | 2.5x head width | 3x+ head width |
| Feet size | = forearm length | +10% | +20-40% |
| Neck length | 0.5 head | 0.3 head | 0.1-0.2 head (or none) |

### Proportion Checklist for Art Direction

- [ ] Head count matches intended character type
- [ ] Navel at ~62% of height (or intentionally displaced)
- [ ] Shoulders width appropriate for personality (wide=strong, narrow=agile)
- [ ] Arms reach mid-thigh when relaxed at sides
- [ ] Elbows align with navel/waist
- [ ] Wrists align with crotch line
- [ ] Knees at midpoint of legs
- [ ] Feet large enough to support the character's visual weight
- [ ] All bony landmarks visible and correctly placed
- [ ] Muscle insertions follow skeleton landmarks
