# Icon Pack Curation

## Quick Reference

| Kriter | Kontrol |
|--------|---------|
| Grid | 24px sistem |
| Stroke | Tek kalınlık ailesi |
| Format | SVG sprite vs font |
| Lisans | MIT / Apache uyumlu |

## Patterns & Decision Matrix

| UI | Öneri |
|----|-------|
| Dashboard | Outline set |
| Oyun HUD | Custom pixel-aligned |

## Code Examples

```text
[ICON_SET] id=heroicons | license=MIT | style=outline | count=~300
```

## Anti-Patterns

- Karışık dolgu + çizgi ikon aynı ekranda.
