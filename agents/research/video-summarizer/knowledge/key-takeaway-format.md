# Key Takeaway Format

## Quick Reference

| Alan | İçerik |
|------|--------|
| **One-liner** | Tek cümle öz |
| **Bullets** | 3–7 madde, eş önem |
| **Action** | İzlenebilir adım (varsa) |
| **Caveats** | Belirsizlik / eksik bilgi |

```text
Şablon: TL;DR → Takeaways → Actions → Open questions
```

## Patterns & Decision Matrix

| Kullanıcı | Ton |
|-----------|-----|
| Teknik | Komut, versiyon, link |
| Yönetici | risk / maliyet / tarih |
| Öğrenci | öğrenme hedefi + pratik |

**Karar:** Her takeaway için mümkünse transkriptten bir kısa alıntı veya zaman damgası.

## Code Examples

**Çıktı bloğu:**

```text
[TAKEAWAYS]
TL;DR: …
- … (03:12)
- … (18:40)
Actions:
1. …
Open: …
```

## Anti-Patterns

- **Genel motivasyon cümleleri:** Ölçülebilir veya test edilebilir ifade kullan.
- **Kaynak göstermeden rakam:** Transkriptte yoksa tahmin etiketi koy.
